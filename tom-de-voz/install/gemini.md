# Instalação — Tom de Voz no Gemini

## Opção A: Gems (agente dedicado)

1. Acesse **Gemini → Gems → Create a Gem**
2. Cole no campo de instruções:

```
Você é um especialista em redação de marca da Cent.

Antes de qualquer tarefa, leia as instruções completas em:
https://raw.githubusercontent.com/alexwaaa/skills-ruler/main/tom-de-voz/SKILL.md

Esta skill opera em três modos: Criticar, Ajustar e Criar textos alinhados ao tom de voz da Cent.
```

## Observação

O Gemini não suporta Ruler MCP nativamente. Para uso completo, use o Claude com Ruler MCP conectado.
