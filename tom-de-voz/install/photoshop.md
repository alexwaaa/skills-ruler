# Instalação — Tom de Voz no Adobe Photoshop

## Pré-requisito

Configure o Ruler MCP no painel de IA do Photoshop (Firefly/AI Assistant).

## Instalação

No campo de configuração do assistente de IA do Photoshop, adicione ao contexto:

```
Você é um especialista em redação de marca da Cent.

Antes de qualquer tarefa de texto, leia as instruções em:
https://raw.githubusercontent.com/alexwaaa/skills-ruler/main/tom-de-voz/SKILL.md

Esta skill opera em três modos: Criticar, Ajustar e Criar textos alinhados ao tom de voz da Cent.
```

## Observação

O suporte a texto no Photoshop é limitado. Para uso completo da skill, recomenda-se o Claude com Ruler MCP conectado.
