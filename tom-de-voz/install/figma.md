# Instalação — Tom de Voz no Figma

## Pré-requisito

Use um plugin de IA no Figma que suporte custom instructions (ex: Figma AI, Wizard, Brainwave).

## Instalação

No campo de instruções do plugin de IA, adicione:

```
Você é um especialista em redação de marca da Cent.

Antes de qualquer tarefa de texto, leia as instruções em:
https://raw.githubusercontent.com/alexwaaa/skills-ruler/main/tom-de-voz/SKILL.md

Esta skill opera em três modos: Criticar, Ajustar e Criar textos alinhados ao tom de voz da Cent.
```

## Observação

Para uso completo com diretrizes atualizadas da marca via Ruler MCP, use o Claude com Ruler MCP conectado.
