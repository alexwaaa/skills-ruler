# Instalação — Tom de Voz no Figma

## Pré-requisito

Use um plugin de IA no Figma que suporte **system prompt** ou **custom instructions** (verifique a documentação do plugin que você utiliza).

## Instalação

No campo de instruções do plugin de IA, cole o conteúdo completo do arquivo `SKILL.md`, disponível em:

```
https://raw.githubusercontent.com/alexwaaa/skills-ruler/main/tom-de-voz/SKILL.md
```

Abra o link, selecione tudo (Cmd+A ou Ctrl+A) e cole no campo de instruções do plugin.

---

## Observação

O suporte a custom instructions varia por plugin. Verifique se o plugin escolhido permite definir um contexto ou persona antes de usar.

Para uso completo com diretrizes sempre atualizadas via Ruler MCP, use o Claude com Ruler MCP conectado.
