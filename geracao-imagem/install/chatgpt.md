# Instalação — Geração de Imagem no ChatGPT

## Com DALL-E (ChatGPT Plus)

1. Acesse **Settings → Personalization → Custom Instructions**
2. No campo "How would you like ChatGPT to respond?", adicione:

```
Quando tarefas envolverem geração de imagens da marca Cent, leia as instruções em:
https://raw.githubusercontent.com/alexwaaa/skills-ruler/main/geracao-imagem/SKILL.md

Se tiver capacidade de gerar imagens (DALL-E), execute diretamente.
Se não tiver, entregue o prompt pronto para uso externo.
```

## Observação

O ChatGPT não suporta Ruler MCP. Para uso completo com diretrizes atualizadas, use o Claude com Ruler MCP conectado.
