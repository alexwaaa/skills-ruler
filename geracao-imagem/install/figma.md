# Instalação — Geração de Imagem no Figma

## Com plugins de IA (ex: Magician, Figma AI)

1. No Claude (com skill instalada + Ruler MCP ativo):
   > "Crie um prompt para geração de imagem no Figma de [descrição]"

2. O Claude gera o prompt alinhado às diretrizes da Cent

3. No Figma, use o plugin de IA de sua preferência e cole o prompt gerado

## Observação

Para uso completo com diretrizes atualizadas via Ruler MCP, use o Claude com Ruler MCP conectado para gerar os prompts.
