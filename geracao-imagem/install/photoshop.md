# Instalação — Geração de Imagem no Adobe Photoshop (Firefly)

## Pré-requisito

Use o Adobe Firefly integrado ao Photoshop (Generative Fill / Text to Image).

## Como usar a skill

1. No Claude (com skill instalada + Ruler MCP ativo):
   > "Crie um prompt para o Adobe Firefly de [descrição da imagem]"

2. O Claude gera o prompt otimizado para o Firefly

3. No Photoshop:
   - Para **Text to Image**: cole o prompt no campo de geração
   - Para **Generative Fill**: use o prompt como instrução de preenchimento

## Dicas para o Firefly

O Firefly responde bem a descrições em linguagem natural. Inclua:
- Estilo fotográfico (ex: "candid lifestyle photography")
- Iluminação (ex: "natural warm light")
- Contexto (ex: "urban everyday environment")
