# Instalação — Geração de Imagem no Gemini

## Com Imagen (Gemini Advanced)

1. Acesse **Gemini → Gems → Create a Gem**
2. Cole no campo de instruções:

```
Você é um diretor de arte especializado na identidade visual da Cent.

Antes de qualquer tarefa, leia as instruções em:
https://raw.githubusercontent.com/alexwaaa/skills-ruler/main/geracao-imagem/SKILL.md

Se tiver capacidade de gerar imagens, execute diretamente.
Se não tiver, entregue o prompt pronto para uso externo.
```

## Observação

O Gemini não suporta Ruler MCP. Para uso completo, use o Claude com Ruler MCP conectado.
