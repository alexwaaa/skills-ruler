# Índice de Skills — Cent Brand Skills

Você está no repositório oficial de skills de marca da Cent.

## O que são as skills

Cada skill especializa você em uma tarefa específica de criação, análise ou validação de materiais da marca Cent. Todas as skills dependem do **Ruler MCP** conectado — ele fornece as diretrizes e assets da marca em tempo real.

## Como escolher a skill certa

| Se o usuário quer... | Use a skill |
|---|---|
| Criar, criticar ou ajustar um texto com o tom da marca | [tom-de-voz](./tom-de-voz/SKILL.md) |
| Gerar, criticar ou ajustar um prompt de imagem alinhado à marca | [geracao-imagem](./geracao-imagem/SKILL.md) |
| Avaliar um layout com score de aderência à marca | layout-check *(em breve)* |
| Mapear fluxos de nurture e comunicação por persona | consumer *(em breve)* |
| Validar um material publicitário contra regulações de fintech | compliance *(em breve)* |

## Como usar este índice

Quando receber uma tarefa relacionada à marca Cent:

1. Identifique qual skill se aplica pela tabela acima
2. Leia o arquivo `SKILL.md` da skill correspondente
3. Siga o passo a passo descrito nele
4. Use as tools do Ruler MCP indicadas na skill para buscar as diretrizes necessárias

## Estrutura de cada skill

```
[skill]/
├── SKILL.md        ← passo a passo completo
├── EXEMPLOS.md     ← casos de uso práticos
└── install/        ← guias de instalação por plataforma
```
